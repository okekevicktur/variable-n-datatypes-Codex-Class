What are variables

Variables are containers for storing data values. In JavaScript, variables are used to hold data that can be changed during the execution of a program.

You can create a variable using the var, let, or const keywords. The var keyword is used to declare a variable that can be re-assigned, while let is used for block-scoped variables, and const is used for constants that cannot be re-assigned.

    let age = 30; // Number
    const isStudent = true; // Boolean

What are datatypes

Datatypes are the classification of data items. In JavaScript, there are two types of datatypes: primitive and non-primitive.

Primitive datatypes include string, number, boolean, null, undefined, bigint, and symbol. Non-primitive datatypes include objects and arrays.

Types of datatypes

Primitive Datatypes

  • String: Represents a sequence of characters.
  • Number: Represents numeric values.
  • Boolean: Represents true or false values.
  • Null: Represents an intentional absence of any value.
  • Undefined: Represents a variable that has been declared but not assigned a value.
  • BigInt: Represents integers with arbitrary precision.
  • Symbol: Represents a unique identifier.

Non-Primitive Datatypes

  • Object: A collection of properties and methods.
  • Array: A special type of object used to store multiple values in a single variable.

Conclusion

Understanding variables and datatypes is crucial for programming in JavaScript. Variables allow you to store and manipulate data, while datatypes help you understand the kind of data you are working with.
